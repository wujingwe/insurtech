# 隱私權政策 (Privacy Policy)

**最後更新日期：2025年12月28日**

歡迎使用 **「保險存摺保單顯示器」**（以下簡稱「本擴充功能」）。我們非常重視您的隱私，本政策旨在說明我們如何處理您在使用過程中的資訊。

## 1. 核心隱私原則
本擴充功能採取 **「本地優先 (Local-first)」** 的設計架構：
* **不收集資料**：我們不會收集、儲存或傳送您的任何個人保險資料、登入憑證或瀏覽行為。
* **無伺服器通信**：本擴充功能不具備後端伺服器，所有邏輯均在您的瀏覽器本地端執行。
* **資料不轉售**：由於我們不收集資料，因此絕不會將您的資訊提供或轉售給任何第三方。

## 2. 權限使用說明
為了移除遮罩與顯示保單內容，本擴充功能僅申請必要的最小權限：
* **`scripting`**：僅用於在您主動打開「保險存摺」網頁時，暫時性地修改網頁樣式以移除遮罩。
* **`webRequest`**：向[保險存摺](https://insurtech.lia-roc.org.tw/lia-passbook-server/api/ipb/)存取保單內容。

## 3. 第三方網站聲明
本擴充功能僅針對「壽險公會保險存摺」官方網站提供介面優化服務。當您透過該網站查詢保單時，您的資料仍受該官方網站之隱私權政策保護。我們建議您同時閱讀官方網站的相關聲明。

## 4. 安全性
我們透過 Manifest V3 規範開發，嚴格禁止執行任何遠端代碼 (Remote Code)，確保擴充功能在受信任的環境下執行。
